# Dataset Information

## Required Datasets
All datasets required for training the models are hosted on Kaggle:

1. **Diabetes & GI Prediction Dataset**
   - Source: [Cleaned Data 2](https://www.kaggle.com/datasets/aarogyabhandari/cleaned-data-2)
   - Used for: Training diabetes prediction and glycemic index models
   

2. **Nutritional Label Dataset**
   - Source: [Nutritional Label Data](https://www.kaggle.com/datasets/aarogyabhandari/nutritional-label-data)
   - Used for: Training YOLOv8 nutritional label detection model
